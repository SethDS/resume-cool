VentureBound is a full-stack application for adventurers. It allows the user to create custom adventures and save all info about them, including Google Maps coordinates and multiple images.
  • REST API built in Node handles requests to: log in with local authentication, create, store and view adventures
  • Google Maps API integration allows users to drag and adjust for exact adventure location, and view all adventure location markers on a map
  • All adventure images are converted to base 64, sent to the back end and stored in SQL database and then pulled and displayed alongside adventure location map
